# submissions

수강생은 **`본인GitHub아이디`** 이름의 폴더를 만들고, 그 안에 **단일 프로젝트 폴더 1개**를 넣습니다.

예: GitHub 아이디가 `hong` 이면:

```
submissions/hong/todo-app/     ← Spring Boot 루트
submissions/hong/todo-app/frontend/   ← React (node_modules 제외)
```

`submissions/hong/backend/` 와 `submissions/hong/frontend/` 처럼 **나누지 마세요.**

이 폴더 아래에는 **본인 폴더만** 추가하세요. 다른 수강생 폴더는 수정하지 않습니다.
